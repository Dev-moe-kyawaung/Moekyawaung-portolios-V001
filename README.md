## About

This repository contains a production-ready multi-page portfolio website for **Moe Kyaw Aung** with:
- Fixed top navigation
- Animated hero background
- Dark / light mode
- Responsive hamburger menu
- Scroll animations and parallax
- Contact form with Formspree integration
- App collection, certificate collection, links, gallery, pricing, FAQ, and Burmese support pages
