# Moe-Kyaw-Aung-Portfolio V.001

Glasspunk multi-page portfolio for **Moe Kyaw Aung**, Senior Android Developer.

## Features
- Responsive Bootstrap 5 layout
- Animated hero and parallax effects
- Dark / light mode
- Contact form with Formspree
- App collection, certificates, links, gallery, pricing, FAQ, and Burmese support pages

## Tech Stack
- HTML5
- Bootstrap 5
- CSS variables
- Vanilla JavaScript
- Font Awesome
- Google Fonts

## Folder Structure
```text
assets/
  css/style.css
  js/main.js
index.html
features.html
apps.html
certificates.html
links.html
gallery.html
pricing.html
faq.html
contact.html
burmese.html
```
